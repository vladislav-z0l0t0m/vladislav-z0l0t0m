### 🚀 Featured Project

**[Mafia Club Hub (Telegram Mini App & Web Platform)](#)** An ecosystem for offline Mafia clubs, transforming traditional paper-and-excel game tracking into a gamified social network inside Telegram.

* **Architecture:** Microservices with **Nest.js**, Node.js, and Event-Driven architecture (**RabbitMQ**) for asynchronous event processing.
* **Frontend:** **React**, TypeScript, Telegram Mini Apps SDK, and **Material UI** for pixel-perfect, mobile-first responsive design.
* **Database & DevOps:** **PostgreSQL** (Core data), **Redis** (Caching/Real-time sessions), **Docker & Docker Compose** for seamless containerization and deployment.

#### 🌟 Key Features & Business Logic:
* **Telegram-Native Authentication:** Seamless, zero-friction user onboarding using Telegram WebApps API (automatically capturing profiles, nicknames, and avatars).
* **Gamified Player Profiles:** Rich, premium-style profiles displaying custom player stats, achievements, unlockable badges, and virtual trophies/medals won in tournaments.
* **Advanced Referee Dashboard:** A high-performance web interface designed for game judges to seamlessly input real-time match data (points, fouls, voting rounds, and player status).
* **Game History Log:** Detailed chronological game logs tracking event sequences (e.g., who was eliminated, shot, or voted out in specific rounds).
* **Viral Tournament Invitations:** In-app friend referral and invitation mechanics, allowing users to invite their Telegram contacts directly to upcoming club tournaments.
* **Automated Bot Notifications:** Real-time event notifications via a dedicated Telegram Bot, alerting users about game schedules, invitations, and rating updates.
